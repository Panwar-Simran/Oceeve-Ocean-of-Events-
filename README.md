## 📌 About the Project

**Oceeve** is a web-based event management portal designed for universities to list, manage, and promote student events — and for students to discover, register, and track their event participation.

Built under a **24-hour sprint** at the Ramanujan Tech Sprint Hackathon, the portal simulates a real-world event management system with separate flows for students and university administrators.

---

## ✨ Features

### 🎓 Student Side
- Browse all upcoming university events from the home page
- View detailed information for each event (schedule, venue, description)
- Register for events via a student registration form
- Login via modal-based authentication
- View and manage registered events from **My Events** page


### 🏫 University Side
- Separate university registration and login flow
- Manage and list events on the portal

### 📄 General
- About page with portal information
- Contact page
- Downloadable event brochure (PDF)
- Responsive design with consistent footer across pages
